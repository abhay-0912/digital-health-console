# Frontend

React + Vite + Tailwind CSS frontend for Digital Health Console.

Setup

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open `http://localhost:3000` in browser

The frontend will connect to the backend at `http://localhost:4000`.

Run both backend and frontend together from the project root:

```powershell
cd "c:\Users\absri\Desktop\sanjeev project"
npm install
npm run install:all
npm run dev
```

Seed demo users then start both:
```powershell
cd "c:\Users\absri\Desktop\sanjeev project"
npm run install:all
npm run dev:seed
```

Demo accounts:
- Doctor: doctor@example.com / Doctor123!
- Patient: patient@example.com / Patient123!
